## 📖 Description

**REST Client App** is a lightweight Postman-like tool built with modern React technologies.  
It allows users to make and manage RESTful API requests, view responses, generate code snippets, and track history — all within a secure, team-oriented environment.

## ✨ Features

- 🔑 **Authentication:** Secure login with NextAuth.js (email/password).
- 🌐 **REST Client:** Request editor with method selector, headers, body, and response viewer.
- 📜 **Generated Code:** Auto-generate request snippets in curl, JS, Python, Java, C#, Go.
- 📂 **History & Analytics:** Store and view previous requests with performance metrics.
- ⚙️ **Variables:** Save and reuse request variables across sessions.
- 🌍 **Internationalization:** Multi-language support (i18n).
- 🖥 **Modern UI:** Responsive interface with TailwindCSS + Shadcn UI.
- 🧪 **Testing & Quality:** Jest, ESLint, Prettier, Husky hooks, Commitlint.

## 🛠️ Technology Stack

- **Next.js (App Router)** — Fullstack React framework
- **React 18+** — UI library
- **TypeScript** — Type-safe JavaScript
- **TailwindCSS** — Utility-first styling
- **Shadcn UI** — Component library on top of Radix & Tailwind
- **NextAuth.js** — Authentication
- **Prisma + MongoDB** — Database and ORM
- **Jest** — Testing framework
- **ESLint + Prettier + Husky + Commitlint** — Code quality tools

## 🚀 Getting Started

### Clone the repository and install dependencies:

```bash
git clone https://github.com/Rasimaru/rest-client-app.git
cd rest-client-app
npm install
```

### Running the Application

Start the development server:

```bash
npm run dev
```

The application will be available at http://localhost:3000.

## 📜 Available Scripts

- `npm run dev`: 🚀 Start development server.
- `npm run build`: 📦 Compiles the application for production, optimizing performance and assets.

- `npm run test`: 🧪 Runs the test suite to ensure the application works as expected.
- `npm run test:coverage`: 📊 Runs tests and generates a code coverage report.
